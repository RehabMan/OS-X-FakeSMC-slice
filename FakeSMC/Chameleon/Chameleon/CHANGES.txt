Changes for Chameleon2 RC4 by Rekursor,  Version 2
======================================================

1. Integration of PCEFI 10.5 automatic detection of PciRoot value 
for gfx cards that have a PciRoot different from 0. 
While it fixes the non automatically working GraphicsEnabler function 
for many cards when PciRoot>0, it it still compatible with the new 
PciRoot override, so users can use this new features if necessary.

2. Added a new com.apple.Boot.plist flag called DeviceID.
Use: "DeviceID=<my_fixed_system_id>" where <my_fixed_system_id> is the UUID 
of your choice as opposed to a random one. 
This option should help users to avoid new registration for pre-installed 
products like iTunes, LogicPro and others.

Sources and binaries (in sym/i386) are both contained in this tarball.

Enjoy!
